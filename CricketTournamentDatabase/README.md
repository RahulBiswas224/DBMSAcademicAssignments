# Cricket Tournament Database (DBMS)

This repository contains the SQL implementation for the Cricket Tournament database assignment for the 3rd Semester BCA course (BCA37107) at Brainware University.

## Project Overview
The database manages match details, player information, bowling statistics, and batting statistics.

## Database Schema
- [cite_start]**MATCH**: Stores details about match venues, dates, and winners. [cite: 7, 21]
- [cite_start]**PLAYER**: Stores personal information of the cricketers. [cite: 9, 23, 24]
- [cite_start]**BOWLING**: Tracks bowling performance (overs, maidens, wickets). [cite: 11, 32]
- [cite_start]**BATTING**: Tracks runs scored by players in specific matches. [cite: 12, 13, 14, 15, 32]

## How to use
1. Create a database in your SQL environment.
2. Run `CricketTournament.sql` to create tables and insert the assignment data.
3. Use the queries provided in your assignment to retrieve statistics.
