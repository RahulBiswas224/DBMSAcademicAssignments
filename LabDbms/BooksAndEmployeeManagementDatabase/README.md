# Books and Employee Management System

This repository contains the SQL and PL/SQL implementation for the **Books and Employee Management System** assignment[cite: 2]. This project was developed for the **Database Management System (BCA37107)** course as part of the 3rd Semester academic curriculum[cite: 2].

## Project Overview
The assignment focuses on creating robust database schemas and implementing business logic using PL/SQL[cite: 2]. Key tasks include managing book costs, handling employee records, and implementing automated database triggers[cite: 2].

## Files Included
* **`BooksAndEmployeeManagement.sql`**: Contains the DDL and DML commands to create `BOOKS_COPY` and `EMPLOYEE` tables and populate them with initial data[cite: 2].
* **PL/SQL Tasks (`Task03` to `Task09`)**:
    * **`Task03_PrimeCheck.sql`**: Logic to determine if a number is prime[cite: 2].
    * **`Task04_Pattern.sql`**: Program to generate a star pattern[cite: 2].
    * **`Task05_UpdateCost.sql`**: Conditional update logic for book costs[cite: 2].
    * **`Task06_DisplayWords.sql`**: String processing to display words on separate lines[cite: 2].
    * **`Task07_UpdateRowCount.sql`**: Mass cost updates with implicit cursor usage[cite: 2].
    * **`Task08_EmpDetails.sql`**: Data retrieval based on user input[cite: 2].
    * **`Task09_Trigger.sql`**: Database trigger to enforce time-based security restrictions[cite: 2].

## Database Schema Highlights
* **BOOKS_COPY**: Maintains book details including unique IDs, titles, and costs[cite: 2].
* **EMPLOYEE**: Stores employee information, including departments and salary data[cite: 2].

## Prerequisites
* An Oracle SQL environment (or compatible RDBMS)[cite: 2].
* `SET SERVEROUTPUT ON;` must be enabled to view console outputs from PL/SQL programs[cite: 2].
